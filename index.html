Projetos

API WhatsApp - Node.js

Essa API pública, feita em Node.js, aceita somente o verbo GET, que simula o WhatsApp. Dentre as funcionalidades da API estão:

  • Listar todos os dados disponíveis na API;
  • Consultar informações de um usuário específico;
  • Listar contatos salvos por um usuário
  • Ver todas as mensagens enviadas e recebidas por um usuário
  • Listar mensagens trocadas entre dois números específicos
  • Buscar mensagens por palavra-chave

Node.js Express JSON

API locadora de filmes - Node.js

Esta API, feita em Node.js, aceita diversos verbos, sendo eles: GET, POST, PUT e DELETE. Ela é basicamente uma API de CRUD de Filme, e não só dele, de outras entidades que o compõem. Dentre as funcionalidades da API, temos:

  • Listar todos os dados
  • Buscar um filme, ou outra entidade, por ID
  • Inserir um filme ou outra entidade
  • Atualizar um filme ou outra entidade
  • Deletar um filme ou outra entidade

Node.js Express Prisma MySQL SQL
